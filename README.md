# REACT-MP-2020

### scripts

- $ npm install 
- **Dev build:** 
$ npm run build:dev
- **Prod build:** 
$ npm run build:prod
- **Start Dev-Server on dev:**
$ npm start:dev
open http://localhost:9000
- **Start Dev-Server on prod:**
$ npm start:prod
open http://localhost:8000
- **Run unit test:**
$ npm run test
- **Run unit test with coverage:**
$ npm run coverage
